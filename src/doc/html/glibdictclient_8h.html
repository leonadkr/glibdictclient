This header file includes error codes and function primitives.

Typical use of this class:

DictClient *dict_client;
gchar *response;
dict_client = dict_client_new();
dict_client_connect( dict_client, "localhost", 2628, NULL, NULL, NULL );
response = dict_client_show_server( dict_client, NULL );
g_print( "%s\n", response );
g_free( response );
dict_client_disconnect( dict_client, NULL, NULL );
g_object_unref( G_OBJECT( dict_client ) );

Looks up the word in the database of the server.

anchor dict_client_define

If database is ! , then all databases of the server will be scanned until the first match. If database is * , then all databases of the server will be scaned for all matches. The database is searched in the same order as that got by dict_client_show_databases(). Sizes of words, databases, descriptions and definitions are the same.

Parameters
[in]selfA DictClient instance.
[in]databaseA database to search in, must not be NULL.
[in]wordA word to search, must not be NULL.
[out]wordsIf not NULL, holds an array of words found in the databases.
[out]databasesIf not NULL, holds an array of the databases holding the words.
[out]descriptionsIf not NULL, holds an array of the descriptions about the databases.
[out]definitionsIf not NULL, holds an array of the definitions of the words.
[out]errorIf not NULL and an error occured, holds a newly allocated GError instance.
Returns
A number of the found definitions or -1 on error.

◆ dict_client_match()

glong dict_client_match ( DictClient * self,
const gchar * database,
const gchar * strategy,
const gchar * word,
GStrv * databases,
GStrv * words,
GError ** error )

Trys to match the word in the database with the selected strategy.

This function scans an index of the database and returns the words array holding words those were found using the strategy. Not all databases support all strategies, but all databases must support prefix and exact. Sizes of returned databases and words are the same. Use items of databases and words as pairs.

Parameters
[in]selfA DictClient instance.
[in]databaseA database to search in, must not be NULL.
[in]strategyA strategy to search with, must not be NULL.
[in]wordA word to search, must not be NULL.
[out]databasesIf not NULL, holds an array of the databases holding the words. May be NUll, if no word in database with strategy.
[out]wordsIf not NULL, holds an array of words found in the databases. May be NUll, if no word in database with strategy.
[out]errorIf not NULL and an error occured, holds a newly allocated GError instance.
Returns
A number of the found database-word pairs or -1 on error.
